# 01_架構總覽與導讀

## 先理解這件事

你現在用的是一個「本地端 coding agent」，核心是：

1. 你可以用白話中文下指令
2. 系統先分析、再規劃、再執行
3. 修改一定走安全流程（可驗證、可回滾）

主線程式在 `repo_guardian_mcp/`。

---

## 你最常用的命令

```bash
uv run repo-guardian chat .
uv run repo-guardian run . --prompt "請分析這個專案" --task-type analyze
uv run repo-guardian run . --prompt "幫我修改 README" --task-type edit
```

---

## 你每天建議這樣用

1. 上工前：`ops run --profile day-start`
2. 開發中：`chat` / `run` / `diff` / `rollback`
3. 收工前：`ops run --profile day-end`

---
