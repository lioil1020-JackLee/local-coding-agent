## 📌 建議閱讀順序（務必照順序）

1. README.md（快速理解專案）
2. docs/目標與方向.md（了解為什麼做這個專案）
3. docs/設計方案.md（理解整體架構）
4. docs/工作流程與Continue設定.md（理解實際運作方式）
5. docs/變更與進度.md（了解目前做到哪）
6. docs/合作與貢獻指南.md（如何參與開發）
